# Roadmap

## v0.1.0-beta.1 — local monitoring foundation

- [x] Versioned event model and deterministic state reducer
- [x] Non-blocking Hook helper and local queue
- [x] Safe Hook preview, install, backup, verification, and uninstall
- [x] Local SQLite history and privacy boundary
- [x] Bilingual task monitor, settings, diagnostics, onboarding, sounds, and notifications
- [x] Opt-in fail-closed transcript watcher
- [x] macOS ARM64 and Windows x64 release automation
- [ ] Real-device acceptance on macOS 13+ Apple Silicon
- [ ] Real-device acceptance on Windows 11 x64
- [ ] Publish signed-off preview installers

## Later

- Apple Developer ID signing and notarization
- Windows Authenticode signing
- Secure automatic updates
- Optional App Server task management after a separate security review
- Additional platform targets based on maintainer capacity

Out of scope until explicitly approved: cloud accounts, telemetry, conversation-content storage, or controlling approval/input from this application.
